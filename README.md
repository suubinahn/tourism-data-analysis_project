## mulcam_tourism-data-analysis_project
## [파이널 프로젝트] 대중교통 데이터를 활용한 관광명소 분석 프로젝트
## 프로젝트 소개
대중교통 이용 데이터를 기반으로 주요 관광지에 집중된 방문 수요를 분산하고,  
숨은 관광명소를 발굴하여 관광객 맞춤형 추천 및 투어버스 노선을 제안하는 데이터 분석 프로젝트입니다.
## 프로젝트 목표
- 대중교통 이용량 데이터를 활용한 관광지 접근성 분석
- 지역별 숨은 관광명소 발굴
- 관광객 분산 및 오버투어리즘 완화 방안 제안
- 지역 맞춤형 투어버스 노선 설계 및 여행 추천 서비스 기획
## 프로젝트 진행 순서
- 프로젝트 배경 분석  
- 데이터 수집 및 전처리  
- 자체 평가지표 개발  
- 지역별 대중교통 특성 EDA  
- 투어버스 노선 설계  
- 여행지 추천 서비스 기획
## 사용 데이터
- 대한민국 구석구석 관광 데이터
- 한국관광 데이터랩
- 교통카드 빅데이터 시스템
- 네이버 지도
- 포털사이트 리뷰 데이터
## 주요 분석 내용
- 지역별 총 승객 수 / 총 노선 수 비교
- 총 환승 수 / 총 소요시간 / 총 도보시간 비교
- 숨은 관광명소와 일반 관광지 간 접근성 차이 분석
- 지역별 관광 패턴 및 이동 특성 비교
## 결과 및 인사이트
- 대중교통 접근성이 낮지만 잠재력 있는 숨은 관광명소 발굴
- 지역 맞춤형 투어버스 노선 제안
- 관광객 분산 및 지역 관광 활성화 전략 제시
- 여행 추천 서비스 적용 가능성 검증
